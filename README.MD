# Install
## Windows
https://gist.github.com/hunterlawson/acf4f65f30867d1642926ad91bc9839d

Ensure `HOME` environment variable is set to `C:\Users\<user_name>`
Emacs config dir will then be under `%HOME%\.emacs.d`
